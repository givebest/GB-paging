SEO支持

客户端访问URL：
http://www.example.com/ajax.html#!key=value
Googlebot 抓取URL(需服务器端支持)：
http://www.example.com/ajax.html?_escaped_fragment_=key=value

详细：https://developers.google.com/webmasters/ajax-crawling/docs/getting-started

不过，Google 已经不建议使用了，详见： https://webmasters.googleblog.com/2015/10/deprecating-our-ajax-crawling-scheme.html.

  1. Google不建议使用上述方法，只要站点允许 Googlebot 抓取 CSS 和 JavaScript, 即可抓取到内容。
  2. 建设使用 HTML5 中的新 History API pushState 体验会更好。

gbPaging({
    total: 101,
    paramName: 'p',
    curPage: getPage('p', '!#'),
    size: 5,
    isSEO: true
});

辅助函数

function getPage(pname, other){
    var page = window.location.search.replace( other + pname + '=', '') || 1;
    return parseInt(page);
}
